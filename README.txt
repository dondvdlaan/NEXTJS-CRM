NEXTJS-CRM

This simple CRM application, written in Next.js, has been transformed from the 
original at https://github.com/luisccode/crm-nextjs. Typescript, Api's, DRY and 
constants have been added for better understnding of the source code.

Stack:
- Next.js
- Typescript
- Tailwind CSS
- JSON Server

Start-up:
- json-server db.json --port 4000
- User/PW : admin@admin.com / 1234
